# AnimeFavorite

## General info
An application for storing your favorite Anime and Cartoons.

## Install and run
To install the project use the command:
```bash
$ npm install 
$ yarn install
```
To start the project use the command:
```bash
$ npm start 
$ yarn start
```
Server run on 
$ localhost:3030
